Aula 02

Evolução do JavaScript

1970 - Nascimento do precessor da internet

1993 - Criação do HTML por Tim Berners-Lee

1994 - Fundação da Netscape e do Mozila

1995 - Criação do Mocha(antecessor do Javascript) e o Java

1997 - JavaScript foi padronizada pela ECMA (devido ao JScript, criado pela microsoft para o internet explorer), nascendo o ECMAScript

2002 - Mozilla se tornou o Firefox

2008 - Google cria o Google Chrome

2009 - criação do motor v8 (motor de execução do JavaScript no Google Chrome)

2010 - Criação do node.js sobre o v8 do Chrome, executando o JavaScript fora do navegador

Aula 04

Escrever "html:5" no VSCode para gerar a base para uma pagina em html5

O html se baseia em tags de marcação, sendo algumas delas:

  • <html></html> que são a base da página
  • <head></head> para o cabeçalho da página
  • <meta> para inserir os metadados da página
  • <title></title> para o titulo dentro do cabeçalho
  • <style></style> para adicionar estilo (CSS) a página
  • <body></body> que é o corpo da página
  • <h1></h1> para titulos grandes
  • <h2></h2> para titulos um pouco menores
  • <p></p> para parágrafos
  • <script></script> para inserir codigos JavaScript ao html

Variáveis

Um sinal de igual(=) se le "recebe"

Quando um valor está atribuido a uma variável ao atribuir um segundo valor o primeiro é perdido

Para esvaziar uma variável é só faze-la receber 'null'

Exemplo de criação de variável

var a = 5

Regras para criação de identificadores (nomes de variável):

  • Podem começar com letras, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Para limpar o console do node ctrl+l

Dicas para criação de nomes de variáveis:

  • Maiúsculas e minúsculas fazem diferença
  • Tente escolher nomes coerentes para as variáveis
  • Evite se tornar um "programador alfabeto" ou um "programador contador"

Tipos de Primitivos

  • number
    • infinity
    • NaN
  • string
  • boolean
  • null
  • undefined
  • object
    • Array
  • function

Para saber o tipo primitivo de uma variável é só usar o comando:

tipeof(var)

ou

tipeof var
